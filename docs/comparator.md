# Interactive Comparator

The interactive comparator lets you pick any two formatting configurations and see a side-by-side diff of the same source file formatted both ways.

**<a href="../compare.html">Open the comparator →</a>**

Choose a Java language level (8, 17, 21, or 25), select two configurations from the dropdowns, and the diff updates instantly. All 48 pre-rendered output files are bundled — no server required.
